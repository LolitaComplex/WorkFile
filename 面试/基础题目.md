顺序表 / 链表的区别
a.内存：顺序表内存空间连续，链表内存空间不连续；顺序表一次开辟永久使用，链表动态开辟；顺序表内存利用率高于链表内存利用率
b.增删改查插：数据增/删/插入链表优于顺序表，数据访问使用顺序表优于链表
c.能结合分析OC语言 或 Java语言分析数组的设计思想

操作系统
进程和线程的区别
a.进程是操作系统分配资源的最小单位；线程是程序执行的最小单元；
b.进程之间互相独立，独占资源空间；进程内线程之间共享程序的内存空间
c.资源开销：进程资源开销大，线程资源开销小

HTTP / HTTPS 区别
a.证书: HTTPS需要证书，会产生证书费用，HTTP免费
b.安全：HTTPS秘文传输，HTTP明文传输；
c.端口：HTTP使用80端口，HTTPS使用443端口
d.了解HTTPS的加密验证过程，或RSA加密流程
